## 🚀 Структура проєкту

Проєкт розбитий на окремі директорії для кожної лабораторної роботи. Кожна папка містить власний `index.html` для зручної навігації між скриптами.

* [cite_start]**LAB01: Налаштування середовища PHP розробки** [cite: 13]
  * [cite_start]Обчислення добутку чисел від 1 до 10 із заміром часу виконання скрипта[cite: 20].
  * [cite_start]Робота з арифметичними операторами (залишок від ділення та ціла частина)[cite: 21, 46].
* [cite_start]**LAB02: Основи роботи з PHP** [cite: 89]
  * [cite_start]Обчислення площі сектора круга з використанням вбудованих математичних функцій `deg2rad()` та `pow()`[cite: 93, 112].
  * [cite_start]Робота з булевими типами даних та їх явне перетворення (`integer`)[cite: 114].

## ⚙️ Як запустити локально
Оскільки репозиторій містить бекенд-код (PHP), відкриття HTML файлів безпосередньо у браузері не запустить скрипти. 
1. [cite_start]Встановіть локальний веб-сервер (наприклад, OpenServer або XAMPP)[cite: 15, 60].
2. Клонуйте цей репозиторій у папку доменів вашого сервера (наприклад, `C:\OSPanel\domains\phplabs.local`).
3. Запустіть сервер та відкрийте у браузері `http://phplabs.local/LAB01/` або `http://phplabs.local/LAB02/`.
