# 🐘 РНР Лабораторні роботи

Колекція лабораторних робіт з дисципліни «Веб-технології та веб-дизайн». 

**Виконавець:** студент 3-го курсу групи KN1-B23, Шафранський Андрій.
**Варіант:** 5.

## 🚀 Структура проєкту

Проєкт розбитий на окремі директорії для кожної лабораторної роботи. Кожна папка містить власний `index.html` для зручної навігації між скриптами.

* **LAB01: Налаштування середовища PHP розробки** 
  * Обчислення добутку чисел від 1 до 10 із заміром часу виконання скрипта.
  * Робота з арифметичними операторами (залишок від ділення та ціла частина).
* **LAB02: Основи роботи з PHP** 
  * Обчислення площі сектора круга з використанням вбудованих математичних функцій `deg2rad()` та `pow()`.
  * Робота з булевими типами даних та їх явне перетворення (`integer`).

## 🛠 Технології та інструменти
* **Мови:** PHP 8.0, HTML5
* **Середовище:** Локальний сервер (Open Server Panel) для виконання PHP-коду.
* **Редактор:** Visual Studio Code.

## ⚙️ Як запустити локально
Оскільки репозиторій містить бекенд-код (PHP), відкриття HTML файлів безпосередньо у браузері не запустить скрипти. 
1. Встановіть локальний веб-сервер (наприклад, OpenServer або XAMPP).
2. Клонуйте цей репозиторій у папку доменів вашого сервера (наприклад, `C:\OSPanel\domains\phplabs.local`).
3. Запустіть сервер та відкрийте у браузері `http://phplabs.local/LAB01/` або `http://phplabs.local/LAB02/`.
